# Banking System

Курсов проект — банкова система на C++.

## Чекпойнт 2 — текущ обхват

Имплементирани:
- Класова йерархия: `Person` (абстрактен базов клас) → `Customer`, `Admin`.
- Клас `Account` (IBAN, баланс, валута, статус).
- Клас `Bank` — централен мениджър с CRUD операции.
- CRUD управление на клиенти (Create / Read / Update / Delete).
- CRUD управление на сметки (Create / Read / Update / Delete).
- Демо меню в `main.cpp`.

Данните се пазят само в паметта (без persistence засега).

## Билд и стартиране

```sh
make
./banking
```

Изисква `g++` с поддръжка на C++17.

## Структура

```
include/   — header файлове (декларации)
src/       — имплементации + main.cpp
Makefile
```

## Бележки спрямо чекпойнт 1

Засега йерархията следва оригиналния план от чекпойнт 1. В следващите чекпойнти ще се добавят:
депозит/теглене, преводи, история на транзакции, извлечения, карти, права.
