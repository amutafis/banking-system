# Banking System

Курсов проект — банкова система на C++.

## Чекпойнт 2 — текущ обхват

Имплементирани:
- Класова йерархия: `Person` (абстрактен базов клас) → `Customer`, `Admin`.
- Клас `Account` (IBAN, баланс, валута, статус).
- Клас `Bank` — централен мениджър с CRUD операции.
- CRUD управление на клиенти (Create / Read / Update / Delete).
- CRUD управление на сметки (Create / Read / Update / Delete).
- Демо меню в `main.cpp`.

Данните се пазят само в паметта (без persistence засега).

## Билд и стартиране

```sh
make
./banking
```

Изисква `g++` с поддръжка на C++17.

## Структура

```
include/   — header файлове (декларации)
src/       — имплементации + main.cpp
Makefile
```

## Меню (главни опции)

| # | Действие |
|---|---|
| 1 | Създай клиент |
| 2 | Покажи всички клиенти |
| 3 | Намери клиент по ID |
| 4 | Редактирай клиент |
| 5 | Изтрий клиент (каскадно премахва и неговите сметки) |
| 6 | Създай сметка |
| 7 | Покажи всички сметки |
| 8 | Намери сметка по IBAN |
| 9 | Промени статус на сметка (ACTIVE / BLOCKED / CLOSED) |
| 10 | Изтрий сметка |
| 11 | Покажи сметките на конкретен клиент |
| 0 | Изход |

При стартиране се зареждат двама демо клиенти и две сметки, за да може менюто да се тества веднага.
